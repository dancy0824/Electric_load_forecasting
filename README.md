# Electric_load_forecasting
This is an electric load forecasting practice using Python.
The input files include a csv file of multiple years' hourly electric load for multiple zones and a csv file of multiple years' hourly temperature from mutiple weather stations.
This practice is a demo, to show how to use Python programming for the data analytics whole process of importing data, data manuplation, exploratorydata analysis, data visualzation, modeling, make prediction and model evaluation.
The main packages used: pandas, matplotlib, and statsmodels.
